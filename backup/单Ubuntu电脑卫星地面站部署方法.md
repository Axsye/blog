# 1. 硬件配置

电台：IC-9700 USB接IF输出到电脑，同时接USB转CIV到9700的CIV接口
旋转器：G5500 控制接口 GS-2320B 通过USB连接到电脑
电脑配置：i3-5015U 8GB RAM 128GB SSD
